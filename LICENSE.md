## Preamble

This License is modeled on the Apache License, Version 2.0, and is adapted to address the unique characteristics of large language models (LLMs), including training data, model weights, fine-tuned derivatives, distilled models, and inference outputs.

Where Apache 2.0 governs source code and compiled artifacts, this License governs **LLM Artifacts**: datasets, model weights, system prompts, evaluation benchmarks, embedding corpora, and similar materials that are primarily consumed, generated by, or used to train language models.

The goals of this License are:
1. Permissive use — including for training, fine-tuning, distillation, and inference.
2. Attribution that is meaningful in the context of LLM pipelines.
3. Explicit clarity on the status of model outputs.
4. Patent non-aggression consistent with Apache 2.0.
